# API Спецификации Kakebo v2.0

> Полная спецификация REST API для системы персонального финансового учёта

---

## 🔐 Аутентификация

Все эндпоинты требуют аутентификации через HttpOnly Cookie:

```typescript
// Middleware для всех Kakebo роутов
kakebo.use('*', requireAuth)
```

## 📊 Базовые эндпоинты (v1.0 — совместимость)

### GET `/api/kakebo`
Получить данные за месяц

**Query Parameters:**
```typescript
{
  year?: number      // По умолчанию: текущий год
  month?: number     // По умолчанию: текущий месяц
}
```

**Response:**
```json
{
  "settings": {
    "monthLimit": 10000,
    "income": 60000
  },
  "entries": [
    {
      "id": "uuid",
      "date": "2026-04-24",
      "categoryId": "uuid",
      "category": {
        "id": "uuid",
        "name": "Жизнь",
        "type": "SYSTEM"
      },
      "description": "Обед",
      "amount": 500,
      "isNecessary": true,
      "isAutoGenerated": false
    }
  ],
  "summary": {
    "totalSpent": 12500,
    "daysInMonth": 30,
    "daysWithEntries": 15,
    "fixedExpenses": 20000,
    "remaining": 35000,
    "byCategory": {
      "sys-life": 7000,
      "sys-culture": 3000,
      "sys-extra": 2000,
      "sys-unexpected": 500
    }
  }
}
```

---

### POST `/api/kakebo`
Создать запись

**Body:**
```typescript
{
  date: string;              // ISO date string "2026-04-24"
  categoryId: string;        // UUID категории
  description: string;       // max 500 chars
  amount: number;            // > 0
  isNecessary?: boolean;     // default: false
}
```

**Response:**
```json
{
  "message": "Запись создана",
  "entry": {
    "id": "uuid",
    "date": "2026-04-24T00:00:00.000Z",
    "categoryId": "uuid",
    "description": "Обед",
    "amount": 500,
    "isNecessary": true
  }
}
```

---

### PUT `/api/kakebo/:id`
Обновить запись

**Body:**
```typescript
{
  date?: string;
  categoryId?: string;
  description?: string;
  amount?: number;
  isNecessary?: boolean;
}
```

**Response:**
```json
{
  "message": "Запись обновлена",
  "entry": { ... }
}
```

---

### DELETE `/api/kakebo/:id`
Удалить запись

**Response:**
```json
{
  "message": "Запись удалена"
}
```

## 📁 Категории (NEW в v2.0)

### GET `/api/kakebo/categories`
Получить все категории пользователя (системные + свои)

**Query Parameters:**
```typescript
{
  type?: 'SYSTEM' | 'CUSTOM';  // Фильтр по типу
  includeChildren?: boolean;    // Включать дочерние категории
}
```

**Response:**
```json
{
  "categories": [
    {
      "id": "sys-life",
      "profileId": null,
      "name": "Жизнь",
      "type": "SYSTEM",
      "parentId": null,
      "isFixed": false,
      "isEssential": true,
      "icon": "Home",
      "color": "#339af0",
      "order": 1,
      "children": [
        {
          "id": "uuid",
          "name": "Транспорт",
          "type": "CUSTOM",
          "parentId": "sys-life",
          "isFixed": false,
          "isEssential": true
        }
      ]
    },
    {
      "id": "uuid",
      "profileId": "uuid",
      "name": "Кафе",
      "type": "CUSTOM",
      "parentId": null,
      "isFixed": false,
      "isEssential": false,
      "icon": "Coffee",
      "color": null,
      "order": 0
    }
  ]
}
```

---

### POST `/api/kakebo/categories`
Создать категорию

**Body:**
```typescript
{
  name: string;           // Название категории
  parentId?: string;      // UUID родительской категории (опционально)
  isFixed?: boolean;      // default: false
  isEssential?: boolean;  // default: true
  icon?: string;          // Имя иконки Lucide
  color?: string;         // Hex цвет (#339af0)
}
```

**Response:**
```json
{
  "message": "Категория создана",
  "category": {
    "id": "uuid",
    "profileId": "uuid",
    "name": "Транспорт",
    "type": "CUSTOM",
    "parentId": null,
    "isFixed": false,
    "isEssential": true,
    "icon": "Car",
    "color": null,
    "order": 0
  }
}
```

**Валидация:**
- Название должно быть уникальным для пользователя
- Нельзя создать подкатегорию для SYSTEM категории без profileId
- Нельзя создать цикл в иерархии (родитель не может быть потомком)

---

### PUT `/api/kakebo/categories/:id`
Обновить категорию

**Body:**
```typescript
{
  name?: string;
  isFixed?: boolean;
  isEssential?: boolean;
  icon?: string;
  color?: string;
}
```

**Ограничения:**
- SYSTEM категории нельзя переименовать или удалить
- Нельзя изменить parentId на собственного потомка

**Response:**
```json
{
  "message": "Категория обновлена",
  "category": { ... }
}
```

---

### DELETE `/api/kakebo/categories/:id`
Удалить категорию

**Ограничения:**
- SYSTEM категории нельзя удалить
- Если есть записи в этой категории:
  - Либо вернуть ошибку 409 с предложением перепривязать
  - Либо автоматически перепривязать к родительской

**Body (опционально):**
```typescript
{
  transferToCategoryId?: string;  // Перепривязать записи к этой категории
}
```

**Response:**
```json
{
  "message": "Категория удалена",
  "entriesTransferred": 5  // Количество перепривязанных записей
}
```

## 🔁 Фиксированные траты (NEW в v2.0)

### GET `/api/kakebo/fixed-expenses`
Получить все активные фиксированные траты пользователя

**Query Parameters:**
```typescript
{
  isActive?: boolean;  // Фильтр по статусу
  categoryId?: string; // Фильтр по категории
}
```

**Response:**
```json
{
  "fixedExpenses": [
    {
      "id": "uuid",
      "profileId": "uuid",
      "categoryId": "uuid",
      "category": {
        "id": "uuid",
        "name": "Жизнь",
        "icon": "Home"
      },
      "description": "Аренда квартиры",
      "amount": 15000,
      "dayOfMonth": 1,
      "isActive": true,
      "startDate": "2026-01-01",
      "endDate": null,
      "notes": null
    }
  ]
}
```

---

### POST `/api/kakebo/fixed-expenses`
Создать фиксированную трату

**Body:**
```typescript
{
  categoryId: string;    // UUID категории (должна быть isFixed = true или установить)
  description: string;   // max 255 chars
  amount: number;        // > 0
  dayOfMonth: number;    // 1-31
  startDate?: string;    // ISO date
  endDate?: string;      // ISO date
  notes?: string;        // max 1000 chars
}
```

**Side Effects:**
- Автоматически создаёт `KakeboEntry` за текущий месяц и следующие 3 месяца
- Устанавливает `isAutoGenerated = true` для созданных записей

**Response:**
```json
{
  "message": "Фиксированная трата создана",
  "fixedExpense": { ... },
  "generatedEntries": [
    { "month": "2026-04", "entryId": "uuid" },
    { "month": "2026-05", "entryId": "uuid" },
    { "month": "2026-06", "entryId": "uuid" },
    { "month": "2026-07", "entryId": "uuid" }
  ]
}
```

---

### PUT `/api/kakebo/fixed-expenses/:id`
Обновить фиксированную трату

**Body:**
```typescript
{
  categoryId?: string;
  description?: string;
  amount?: number;
  dayOfMonth?: number;
  isActive?: boolean;
  endDate?: string;
  notes?: string;
}
```

**Side Effects:**
- При изменении `amount` или `dayOfMonth`: предложить пересоздать будущие записи

**Response:**
```json
{
  "message": "Фиксированная трата обновлена",
  "fixedExpense": { ... }
}
```

---

### DELETE `/api/kakebo/fixed-expenses/:id`
Удалить фиксированную трату

**Опции:**
```typescript
{
  deleteFutureEntries?: boolean;  // Удалить будущие автоматически созданные записи
  keepPastEntries?: boolean;      // Сохранить прошлые записи (default: true)
}
```

**Response:**
```json
{
  "message": "Фиксированная трата удалена",
  "deletedEntriesCount": 4
}
```

## 💰 Бюджетное уравнение (NEW в v2.0)

### GET `/api/kakebo/budget/:year/:month`
Получить бюджет и расчёт уравнения

**Response:**
```json
{
  "budget": {
    "profileId": "uuid",
    "year": 2026,
    "month": 4,
    "income": 60000,
    "createdAt": "2026-04-01T00:00:00.000Z",
    "updatedAt": "2026-04-01T00:00:00.000Z"
  },
  "calculation": {
    "income": 60000,
    "fixedExpenses": 20000,
    "savingGoal": 5000,
    "remaining": 35000,
    "spentOnVariable": 12500,
    "remainingToSpend": 22500,
    "isOverBudget": false
  },
  "warning": null
  // или если превышен:
  // "warning": "Вы превысили бюджет на нефиксированные траты на 5000 у.е."
}
```

**Расчёт:**
```typescript
income = budget.income
fixedExpenses = sum(entries.amount WHERE category.isFixed = true)
savingGoal = goal.targetSave
remaining = income - fixedExpenses - savingGoal
spentOnVariable = sum(entries.amount WHERE category.isFixed = false)
remainingToSpend = remaining - spentOnVariable
isOverBudget = remainingToSpend < 0
```

---

### PUT `/api/kakebo/budget/:year/:month`
Обновить бюджет (доход)

**Body:**
```typescript
{
  income: number;  // > 0
}
```

**Response:**
```json
{
  "message": "Бюджет обновлён",
  "budget": { ... },
  "calculation": { ... }  // Пересчитанный
}
```

## 🚨 Обработка ошибок

### Стандартный формат ошибки
```json
{
  "error": "Сообщение ошибки",
  "code": "ERROR_CODE",
  "details": { ... }  // Опционально
}
```

### Коды ошибок

| Код | HTTP | Описание |
|-----|------|----------|
| `UNAUTHORIZED` | 401 | Требуется авторизация |
| `NOT_FOUND` | 404 | Запись не найдена |
| `FORBIDDEN` | 403 | Нет прав на операцию |
| `VALIDATION_ERROR` | 400 | Ошибка валидации |
| `CONFLICT` | 409 | Конфликт данных (дубликат) |
| `CATEGORY_HAS_ENTRIES` | 409 | Нельзя удалить категорию с записями |
| `INVALID_DATE` | 400 | Неверный формат даты |
| `INVALID_AMOUNT` | 400 | Сумма должна быть > 0 |

### Примеры ошибок

**Категория с записями:**
```json
{
  "error": "Нельзя удалить категорию с записями",
  "code": "CATEGORY_HAS_ENTRIES",
  "details": {
    "entriesCount": 15,
    "transferToCategoryId": null
  }
}
```

**Превышение лимита:**
```json
{
  "error": "Превышен бюджет на нефиксированные траты",
  "code": "OVER_BUDGET",
  "details": {
    "budget": 35000,
    "spent": 40000,
    "overBy": 5000
  }
}
```
