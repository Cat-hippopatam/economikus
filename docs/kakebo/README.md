## 🏗️ Архитектура

### Версии

| Версия | Статус | Описание |
|--------|--------|----------|
| **v1.0** | ✅ Готово | Базовый учёт трат с категориями и лимитами |
| **v2.0** | 🚧 В планах | Полноценная система с целями, фиксированными тратами и дашбордом |

### Технологический стек

- **Backend:** Hono + Prisma ORM + MySQL
- **Frontend:** React 19 + Mantine UI + React Query + Zustand
- **Аутентификация:** HttpOnly Cookie сессии
- **Типы ID:** UUID (в отличие от cuid в примерах документации)

---

## 📊 Ключевые возможности v1.0

- ✅ CRUD записей о тратах
- ✅ 4 предустановленные категории (LIFE, CULTURE, EXTRA, UNEXPECTED)
- ✅ Месячный лимит с визуализацией
- ✅ Рефлексия месяца
- ✅ Адаптивный UI (Mobile-first)
- ✅ Таблица на десктопе / карточки на мобильных

---

## 🚀 Ключевые возможности v2.0 (в разработке)

- 📁 **Гибкие категории** — пользовательские + системные, иерархия
- 🎯 **Цели на месяц** — поведенческие обещания и целевые сбережения
- 🔁 **Фиксированные траты** — автоматическое создание регулярных платежей
- 📐 **Бюджетное уравнение** — Доход − Фиксированные − Цели = Остаток
- 📈 **Дашборд с графиками** — Recharts, срезы, экспорт
