# План тестирования загрузки медиафайлов

## Обзор

Этот документ описывает план тестирования функциональности загрузки медиафайлов для платформы Экономикус. Тестирование охватывает все типы медиафайлов, используемых в курсах и уроках.

**Дата создания:** Март 2026  
**Статус:** Требуется выполнение  
**Приоритет:** Высокий

## 1. Цели тестирования

### 1.1 Основные цели

- ✅ Проверить загрузку всех поддерживаемых типов файлов
- ✅ Проверить валидацию типов и размеров файлов
- ✅ Проверить конвертацию изображений в WebP
- ✅ Проверить работу локального хранилища
- ✅ Проверить корректное отображение загруженных файлов
- ✅ Проверить удаление файлов

### 1.2 Дополнительные цели

- ⏳ Проверить работу CDN хранилища (при наличии)
- ⏳ Проверить обработку ошибок при загрузке
- ⏳ Проверить производительность при загрузке больших файлов

## 2. Область тестирования

### 2.1 Типы медиафайлов

| Тип файла | Категория | Макс. размер | Форматы | Приоритет |
|-----------|-----------|--------------|---------|-----------|
| Аватар пользователя | avatars | 2 MB | JPEG, PNG, GIF, WebP | 🔴 Критический |
| Обложка курса | covers | 5 MB | JPEG, PNG, WebP | 🔴 Критический |
| Обложка урока | covers | 5 MB | JPEG, PNG, WebP | 🔴 Критический |
| Аудио урок | audio | 50 MB | MP3, WAV, OGG | 🟡 Высокий |
| Видео урок | video | 500 MB | MP4, WebM, MOV | 🟡 Высокий |
| Сертификат (изобр.) | certificates | 10 MB | JPEG, PNG, WebP, PDF | 🟢 Средний |
| Сертификат (PDF) | certificates | 10 MB | PDF | 🟢 Средний |

### 2.2 Компоненты для тестирования

#### Frontend компоненты

| Компонент | Файл | Функциональность |
|-----------|------|------------------|

| LessonCoverUpload | `src/components/author/` | Загрузка обложки урока |
| AudioContentEditor | `src/components/author/AudioContentEditor.tsx` | Загрузка аудио |
| VideoContentEditor | `src/components/author/VideoContentEditor.tsx` | Загрузка видео |

#### Backend endpoints

| Endpoint | Метод | Описание | Статус |

## 4. Тестовые данные

### 4.1 Подготовка тестовых файлов

Создайте папку `test-data/media/` со следующими файлами:

```
test-data/media/
├── avatars/
│   ├── avatar_small.jpg       # 100x100, ~10KB
│   ├── avatar_normal.jpg      # 500x500, ~100KB
│   ├── avatar_large.jpg       # 1920x1080, ~500KB
│   ├── avatar_huge.jpg        # 4000x4000, ~3MB (превышение)
│   ├── avatar.png             # PNG с прозрачностью
│   ├── avatar.gif             # Анимированный GIF
│   └── document.pdf           # Для теста недопустимого типа
```

### 4.2 Скрипт генерации тестовых данных

```bash
# Пример создания тестовых изображений (требуется ImageMagick)
convert -size 500x500 xc:blue test-data/media/avatars/avatar_normal.jpg
convert -size 1200x630 xc:green test-data/media/covers/course_cover_1200x630.jpg

# Пример создания тестового аудио (требуется ffmpeg)
ffmpeg -f lavfi -i sine=frequency=440:duration=60 -b:a 128k test-data/media/audio/lesson_short.mp3

# Пример создания тестового видео (требуется ffmpeg)
ffmpeg -f lavfi -i testsrc=duration=60:size=1280x720:rate=30 -c:v libx264 test-data/media/video/lesson_short.mp4
```
