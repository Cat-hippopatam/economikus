## 1. О проекте

**Экономикус** — это образовательная онлайн-платформа для изучения финансов и инвестиций.

### Основные возможности:
- 📚 Курсы по финансам и инвестициям
- 🧮 Финансовые калькуляторы
- 🛠 Инструменты для управления финансами
- 📜 Сертификаты об окончании курсов
- 👤 Личный кабинет с прогрессом обучения
- 💰 Подписки и платежи

### Целевая аудитория:
- Начинающие инвесторы
- Студенты и молодые специалисты
- Все, кто хочет повысить финансовую грамотность

## 2. Структура сайта

### Публичные страницы

| Маршрут | Страница | Описание |
|---------|----------|----------|
| `/` | Главная | Лендинг с курсами и калькуляторами |
| `/catalog` | Каталог курсов | Список всех доступных курсов |
| `/courses/:slug` | Страница курса | Информация о курсе, модули, уроки |
| `/courses/:courseSlug/lessons/:lessonSlug` | Урок | Контент урока (статья/видео/аудио/квиз) |

| `/tools` | Инструменты | Дополнительные инструменты |
| `/postulates` | Постулаты | Базовые финансовые принципы |
| `/info` | Информация | О нас, команда, контакты, новости |
| `/terms` | Условия использования | Правовой документ |
| `/privacy` | Политика конфиденциальности | Правовой документ |

## 3. Технологический стек

### Frontend
- **React 19** — UI фреймворк
- **Vite** — сборщик
- **TypeScript** — типизация
- **Mantine v7** — UI компоненты
- **React Router v6** — маршрутизация
- **TanStack Query** — управление данными
- **Lucide React** — иконки

### Backend
- **Hono** — HTTP фреймворк (Edge-ready)
- **Prisma** — ORM
- **MySQL** — база данных

### Инструменты
- **ESLint** — линтинг
- **Prettier** — форматирование

## 4. Функциональность

### 4.1 Курсы и уроки

**Типы уроков:**
- Article — текстовый урок
- Video — видео урок
- Audio — аудио урок
- Quiz — квиз/тест

**Структура курса:**
```
Курс
├── Модуль 1
│   ├── Урок 1
│   ├── Урок 2
│   └── ...
├── Модуль 2
│   └── ...
└── ...
```

### 4.2 Система подписок

- Бесплатные курсы
- Премиум курсы по подписке
- Индивидуальные подписки

### 4.3 Прогресс обучения

- Отслеживание просмотренных уроков
- Прогресс курса в процентах
- Завершение уроков
- Сертификаты

### 4.4 Избранное и история

- Сохранение уроков в избранное
- История просмотров

### 4.5 Калькуляторы

- Калькулятор сложного процента
- Инвестиционный калькулятор
- Ипотечный калькулятор
- И другие

## 5. Правовые документы

Регулирует отношения между пользователем и платформой:
- Регистрация и аккаунт

### 5.3 Политика cookie (`/cookies`)
Регулирует использование cookies:
- Типы cookies
- Управление cookies
- Сторонние cookies

## 7. Разработка

### Установка

```bash
# Frontend
cd src
npm install
npm run dev

# Backend
cd server
npm install
npm run dev
```

### Структура проекта

```
src/
├── components/       # UI компоненты
├── pages/           # Страницы
├── hooks/           # React хуки
├── services/        # API сервисы
├── constants/       # Константы
└── layouts/         # Макеты страниц

server/
├── routes/          # API роуты
├── middleware/      # Промежуточное ПО
└── index.ts         # Точка входа
```

### Соглашения

- Компоненты: PascalCase
- Хуки: camelCase, префикс `use`
- Константы: UPPER_SNAKE_CASE
- CSS классы: kebab-case
