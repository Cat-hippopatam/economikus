## 1. Карта сайта

### Публичные страницы

```
/
├── /catalog                    — Каталог курсов
├── /courses/:slug              — Страница курса
├── /courses/:courseSlug/lessons/:lessonSlug — Урок
├── /calculators                — Калькуляторы
│   └── /calculators/:slug      — Конкретный калькулятор
├── /tools                      — Инструменты
├── /postulates                 — Постулаты финансов
├── /info                       — О платформе
├── /terms                      — Условия использования
├── /privacy                    — Политика конфиденциальности
├── /cookies                    — Политика cookies
├── /user/:nickname             — Профиль пользователя
├── /login                      — Вход
└── /register                   — Регистрация
```

### Защищённые страницы

```
/profile/settings           — Настройки профиля
/become-author              — Стать автором
```

### Панель автора

```
/author/dashboard           — Дашборд
/author/courses             — Мои курсы
/author/courses/new         — Создать курс
/author/courses/:id         — Редактировать курс
/author/courses/:id/modules — Модули курса
/author/lessons             — Мои уроки
/author/lessons/new         — Создать урок
/author/lessons/:id         — Редактировать урок
/author/analytics           — Аналитика
```

### Админ-панель

```
/admin                      — Дашборд
/admin/courses              — Управление курсами
/admin/lessons              — Управление уроками
/admin/tags                 — Управление тегами
/admin/users                — Управление пользователями
/admin/moderation           — Модерация
/admin/applications         — Заявки авторов
```

## 2. Навигация

### Header (основная навигация)

| Ссылка | URL | Описание |
|--------|-----|----------|
| Каталог | `/catalog` | Каталог курсов |
| Калькуляторы | `/calculators` | Финансовые калькуляторы |
| Инструменты | `/tools` | Дополнительные инструменты |
| Постулаты | `/postulates` | Базовые принципы |
| Информация | `/info` | О платформе |

### Навигация в Header (правая часть)

| Элемент | Описание |
|---------|----------|
| Поиск | Поиск по курсам |
| Аватар/Войти | Меню пользователя или кнопка входа |

### Footer (подвал)

**Платформа:**
- Каталог курсов → `/catalog`
- Калькуляторы → `/calculators`
- Инструменты → `/tools`
- Постулаты → `/postulates`

**Компания:**
- О нас → `/info#about`
- Команда → `/info#team`
- Контакты → `/info#contacts`

**Правовая информация:**
- Условия использования → `/terms`
- Политика конфиденциальности → `/privacy`
- Политика cookie → `/cookies`

## 4. SEO оптимизация

### Текущее состояние

| Элемент | Статус | Комментарий |
|---------|--------|-------------|
| Sitemap.xml | ✅ Готов | `public/sitemap.xml` |
| Robots.txt | ✅ Готов | `public/robots.txt` |
| Человекопонятные URL | ✅ Готов | `/courses/invest-beginners` |
| SSR | ❌ Нет | SPA без серверного рендеринга |
| Schema.org | ❌ Нет | Требует реализации |
| Open Graph | ❌ Нет | Требует реализации |
| Мета-теги | ⚠️ Частично | Только статические в index.html |

### Robots.txt

```txt
User-agent: *
Allow: /
Disallow: /admin
Disallow: /author
Disallow: /api
Disallow: /profile

Sitemap: https://economikus.ru/sitemap.xml
Host: economikus.ru
```

### Roadmap SEO

| Приоритет | Задача |
|-----------|--------|
| Высокий | SSR для индексации (Vike/Next.js) |
| Высокий | Мета-теги на всех страницах |
| Средний | Schema.org микроразметка |
| Средний | Open Graph теги |
| Средний | Оптимизация Core Web Vitals |

### Семантическое ядро

**Высокочастотные:**
- обучение инвестициям
- финансовая грамотность
- онлайн курсы по финансам

**Среднечастотные:**
- как начать инвестировать
- курс инвестиции для начинающих
- финансовый калькулятор онлайн

**Низкочастотные:**
- сложный процент калькулятор
- что такое диверсификация портфеля

## 5. Анимации и интерактивность

### Hero-секции

```css
background: linear-gradient(135deg, #264653 0%, #2A9D8F 100%)
```

### Hover-эффекты карточек

```tsx
// Подъём при наведении
onMouseEnter={(e) => {
  e.currentTarget.style.transform = 'translateY(-4px)'
  e.currentTarget.style.boxShadow = '0 8px 24px rgba(0,0,0,0.12)'
}}
```

### Header при скролле

```tsx
// Sticky с blur
backgroundColor: scrolled ? 'rgba(255, 255, 255, 0.95)' : '#fff',
backdropFilter: scrolled ? 'blur(10px)' : 'none',
transition: 'all 0.2s ease'
```

### Страница 404

Анимированный градиентный текст с `WebkitBackgroundClip: 'text'`

### Переходы

- Время перехода: `0.2s ease`
- Плавные изменения `transform` и `box-shadow`

## 6. Адаптивность

### Breakpoints

| Устройство | Ширина | Особенности |
|------------|--------|-------------|
| Десктоп | ≥1200px | Полный layout с боковыми панелями |
| Планшет | 768-1199px | Компактный layout |
| Мобильный | <768px | Одиночная колонка, выдвижное меню |

### Используемые паттерны

**Grid:**
```tsx
<Grid.Col span={{ base: 12, sm: 6, md: 4 }}>
```

**Stack и Group:**
```tsx
<Stack gap="md">...</Stack>
<Group justify="space-between">...</Group>
```

**Lazy loading:**
```tsx
const HomePage = lazy(() => import('./pages/HomePage'))
<Suspense fallback={<PageLoader />}>
  <HomePage />
</Suspense>
```
