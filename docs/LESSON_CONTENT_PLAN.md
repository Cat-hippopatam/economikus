# План: Модули курсов и Контент уроков

## 1. Текущая архитектура

### Схема БД (уже есть)

```
Course (Курс)
├── id, title, slug, description
├── authorProfileId
└── modules[] → Module

Module (Модуль)
├── id, title, description
├── courseId
├── sortOrder
└── lessons[] → Lesson

Lesson (Урок)
├── id, title, slug, description
├── lessonType (ARTICLE, VIDEO, AUDIO, QUIZ, CALCULATOR)
├── moduleId (опционально)
├── authorProfileId
└── textContent / videoContent / audioContent / quizContent (1:1)
```

### Таблицы контента

| Таблица | Тип урока | Поля |
|---------|-----------|------|
| `TextContent` | ARTICLE | body, wordCount, readingTime |
| `VideoContent` | VIDEO | videoUrl, provider, duration, qualities |
| `AudioContent` | AUDIO | audioUrl, duration |
| `QuizContent` | QUIZ | questions, passingScore |

```
│  Модуль 2: Анализ рынков            │
│  ...                                │
└─────────────────────────────────────┘
```

**API endpoints:**

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `/author/courses/:id/modules` | GET | Список модулей курса |
| `/author/modules` | POST | Создать модуль |
| `/author/modules/:id` | PATCH | Обновить модуль |
| `/author/modules/:id` | DELETE | Удалить модуль |
| `/author/modules/reorder` | POST | Изменить порядок |

## 4. Выбор WYSIWYG редактора

### Варианты

| Редактор | Плюсы | Минусы |
|----------|-------|--------|
| **Tiptap** | Современный, расширяемый, React-friendly | Требует настройки |
| **Quill** | Простой, готовый из коробки | Менее гибкий |
| **Slate** | Полный контроль | Сложный в настройке |

**Рекомендация: Tiptap**
- Хорошая интеграция с React
- Модульная архитектура
- Поддержка Markdown

---

## 5. Загрузка медиа

### Варианты

1. **Прямая загрузка на сервер**
   - Папка `uploads/`
   - Ограничение размера
   - Проблема с большим контентом

2. **S3-совместимое хранилище**
   - MinIO (локально)
   - AWS S3 / Yandex Object Storage
   - Рекомендуется для продакшена

3. **Внешние сервисы**
   - YouTube/Vimeo для видео
   - Только ссылка в БД

**Рекомендация для MVP:**
- Видео: только ссылки на YouTube/Vimeo
- Аудио: загрузка на сервер (до 50MB)
- Изображения: загрузка на сервер (до 2MB)
