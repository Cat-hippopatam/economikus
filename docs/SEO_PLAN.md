## 3. Техническая оптимизация

### 3.1 Приоритет: Критично

#### 3.1.1 Sitemap.xml
```xml
<!-- /public/sitemap.xml -->
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://economikus.ru/</loc>
    <changefreq>daily</changefreq>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://economikus.ru/catalog</loc>
    <changefreq>weekly</changefreq>
    <priority>0.9</priority>
  </url>
  <!-- Все курсы, уроки, страницы -->
</urlset>
```

#### 3.1.2 Robots.txt
```txt
# /public/robots.txt
User-agent: *
Allow: /
Disallow: /admin
Disallow: /author
Disallow: /api
Disallow: /profile

Sitemap: https://economikus.ru/sitemap.xml
Host: economikus.ru
```

### 3.2 Приоритет: Высокий

#### 3.2.1 SSR (Server-Side Rendering)
Рекомендуется внедрить:
- **Вариант A**: Vike (Vite + SSR)
- **Вариант B**: Next.js (миграция)
- **Вариант C**: Prerender (статическая генерация для критических страниц)

#### 3.2.2 Open Graph мета-теги
```tsx
// Компонент SEOHead
<meta property="og:title" content={title} />
<meta property="og:description" content={description} />
<meta property="og:image" content={image} />
<meta property="og:url" content={url} />
<meta property="og:type" content="website" />
<meta name="twitter:card" content="summary_large_image" />
```

#### 3.2.3 Schema.org микроразметка

**Organization:**
```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Экономикус",
  "url": "https://economikus.ru",
  "logo": "https://economikus.ru/logo.png",
  "contactPoint": {
    "@type": "ContactPoint",
    "email": "hello@economikus.ru",
    "contactType": "customer service"
  }
}
```

**Course (для страниц курсов):**
```json
{
  "@context": "https://schema.org",
  "@type": "Course",
  "name": "Название курса",
  "description": "Описание курса",
  "provider": {
    "@type": "Organization",
    "name": "Экономикус"
  }
}
```

**CourseInstance (для уроков):**
```json
{
  "@context": "https://schema.org",
  "@type": "CourseInstance",
  "name": "Название урока",
  "courseMode": "online"
}
```

## 5. Контентная стратегия

### 5.1 Типы контента

| Тип | Частота | Цель |
|-----|---------|------|
| Блог/Статьи | 2/неделя | Трафик, экспертиза |
| Новости | 1/неделя | Свежесть |
| FAQ | 1/месяц | Низкочастотные запросы |
| Гайды | 1/месяц | Средние частотные запросы |

### 5.2 Семантическое ядро

#### Высокочастотные (ВЧ):
- обучение инвестициям
- финансовая грамотность
- онлайн курсы по финансам

#### Среднечастотные (СЧ):
- как начать инвестировать
- курс инвестиции для начинающих
- финансовый калькулятор онлайн

#### Низкочастотные (НЧ):
- как инвестировать в акции новичку
- сложный процент калькулятор
- что такое диверсификация портфеля

### 5.3 Контентный план

1. **Базовые статьи:**
   - Что такое инвестирование: полное руководство
   - Виды инвестиций: акции, облигации, фонды
   - Как создать инвестиционный портфель
   - Риск и доходность: баланс

2. **Калькуляторы (контент):**
   - Калькулятор сложного процента: как посчитать доходность
   - Калькулятор инвестиционного портфеля
   - Калькулятор пенсионных накоплений
