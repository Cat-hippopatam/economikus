## 2. Назначение и область применения

### 2.1 Назначение программного продукта

Программный продукт «Экономикус» представляет собой образовательную веб-платформу, предназначенную для:

1. **Обучения финансовой грамотности** — предоставление пользователям доступа к курсам и урокам по основам финансов, инвестиций и управления личными финансами;

2. **Создания и публикации контента** — обеспечение функциональности для авторов по созданию, редактированию и публикации образовательных материалов;

3. **Управления обучением** — отслеживание прогресса пользователей, ведение истории обучения, выдача сертификатов;

4. **Монетизации контента** — реализация системы подписок для доступа к премиум-контенту;

5. **Социального взаимодействия** — возможность оставлять комментарии, реакции, добавлять материалы в избранное.

### 2.2 Область применения

Программный продукт применяется в следующих областях:

| Область применения | Описание |
|-------------------|----------|
| Образование | Онлайн-обучение финансовой грамотности |
| Профессиональное развитие | Повышение квалификации в области инвестиций |
| Самообразование | Самостоятельное изучение финансовых тем |
| Контент-индустрия | Создание и продажа образовательных курсов |

### 2.3 Класс программы

По ГОСТ 19.101-77 программный продукт относится к классу:

- **Прикладное программное средство**
- **Информационная система**
- **Веб-приложение**

### 2.4 Категории пользователей

| Категория | Описание | Примерное количество |
|-----------|----------|---------------------|
| Студенты | Пользователи, проходящие обучение | Неограниченно |
| Авторы | Создатели образовательного контента | 50-500 |
| Модераторы | Пользователи, модерирующие контент | 5-20 |
| Администраторы | Управление платформой | 2-5 |
